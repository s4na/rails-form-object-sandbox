# Rails Form Object Sandbox

## セットアップ

```
./bin/setup
```

## サーバー起動

```
./bin/webpack-dev-server
```

別ターミナルで

```
./bin/rails s
```

## テスト

```
./bin/bundle exec rspec
```

ex)

```
./bin/bundle exec rspec spec/models/post_spec.rb
```
